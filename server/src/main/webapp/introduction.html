Introduction

Ce TP a pour but de vous introduire les notions de séquences d'instruction et d'alternative en iJava. Il se compose de 3 exercices. Le premier traite le suffixe d'un mot, les autres ne sont qu'une suite inventée pour présenter le déroulement d'un TP à plusieurs exercices.
